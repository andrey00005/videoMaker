# 🎬 ClipMaker – Projeto NLW Rocketseat

![apresentacão do desafio](assets/trailer.gif)

Este projeto foi desenvolvido durante o **NLW Operator** da Rocketseat e tem como objetivo automatizar a criação de vídeos curtos a partir de um vídeo enviado pelo usuário.

A aplicação permite enviar um vídeo, processar o conteúdo utilizando IA e gerar automaticamente um novo vídeo já pronto para download.

---

## 🚀 O que o projeto faz

O usuário pode:

- Enviar um vídeo pela interface
- Processar o vídeo utilizando inteligência artificial
- Gerar automaticamente um novo vídeo com cortes
- Visualizar o resultado diretamente na página
- Fazer o download do vídeo final

---

## 🧠 Tecnologias utilizadas

Durante esse projeto eu trabalhei com tecnologias que ainda não tinha usado na prática, principalmente na parte de integração com API.

Tecnologias usadas:

- HTML
- CSS
- JavaScript
- Tailwind CSS
- Upload de arquivos (vídeo)
- Integração com API de IA
- Código assíncrono (async/await)
- Promises
- Lógica de retry em requisições
- Manipulação do DOM
- GSAP (animações)
- Lucide Icons
